# nadejda.biz.ua

```
git clone https://github.com/tadatuta-studio/nadejda-biz-ua.git
cd nadejda-biz-ua
npm i
./node_modules/.bin/bem server
```

Check http://localhost:8080/desktop.bundles/index/index.html
