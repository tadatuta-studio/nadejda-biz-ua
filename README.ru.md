# nadejda.biz.ua

## Как начать работать с проектом

В консоли выполнить
```
git clone https://github.com/tadatuta-studio/nadejda-biz-ua.git
cd nadejda-biz-ua
npm i
./node_modules/.bin/bem server
```

Открыть в браузере http://localhost:8080/desktop.bundles/index/index.html
